# Frontend

## How do I get started?

Install dependencies / libraries:

```bash
npm ci
```

Then run the frontend...

## How do I run it?

With live-reload (for development):

```bash
npm run dev
```

Without live-reload:

```bash
npm run build && npm run start
```

## Where is the code?

Start in `app/page.tsx`.
